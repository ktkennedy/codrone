# 프로젝트 컨텍스트

## 배포 환경
- 로컬에서 확인하지 않음. git push로 배포 (GitHub Pages)
- 코드 변경 후 로컬 서버 테스트를 제안하지 말 것
- 브라우저에서 직접 실행되는 정적 파일 프로젝트

## 프로젝트 개요
- 초등학생(1-6학년) 대상 드론 코딩 학습 시뮬레이터
- 24주 커리큘럼: 수동 조종 → 미션 → 블록 코딩 → 텍스트 코딩 → 자율비행
- 한국어 전용
- 저학년(1-3학년): 버튼 조작, 15개 미션
- 고학년(4-6학년): 키보드 조작 + 블록/텍스트 코딩, 15개 비행 미션 + 6개 블록 코딩 미션

## 기술 스택
- 순수 HTML/CSS/JavaScript (빌드 도구 없음, npm 없음)
- Three.js 3D 렌더링
- Google Blockly 블록 코딩
- RotorPy 기반 물리 엔진 (250Hz RK4 적분)
- Web Audio API 사운드 합성 (오디오 파일 없음)
- 모든 모듈은 window.DroneSim 네임스페이스 사용

## 프로젝트 구조
- `/shared/` - 공용 모듈 (물리, 카메라, 월드, UI, 미션, 오디오, 센서 등)
- `/elementary-low/` - 저학년 모드
- `/elementary-high/` - 고학년 모드 (coding.html, blockly.html 포함)
- `/materials/` - 수업 자료 (PPT 등)
- `/curriculum.html` - 24주 커리큘럼 대시보드

## 코딩 규칙
- 모든 UI 텍스트는 한국어
- 에셋 파일 의존성 최소화 (프로시저럴 생성 선호)
- script 태그 순서로 의존성 관리 (모듈 번들러 사용하지 않음)
- localStorage로 진행 상황 저장 (서버 불필요)
