기본 조작
drone.takeoff() - 이륙
drone.land() - 착륙
drone.move_forward(거리) - 앞으로 이동 (m)
drone.move_backward(거리) - 뒤로 이동
drone.move_left(거리) - 왼쪽 이동
drone.move_right(거리) - 오른쪽 이동
drone.move_up(높이) - 위로 이동
drone.move_down(높이) - 아래로 이동
drone.turn_left(각도) - 왼쪽 회전 (도)
drone.turn_right(각도) - 오른쪽 회전 (도)
drone.hover(초) - 호버링
drone.wait(초) - 대기

자율비행
drone.fly_to(x, y, z) - 좌표로 자동 비행
drone.return_home() - 홈으로 귀환
drone.follow_path([{"x":0,"y":3,"z":0}, ...]) - 경로 비행
drone.set_speed(속도) - 비행 속도 설정 (1~10)

센서
drone.get_altitude() - 현재 높이
drone.get_speed() - 현재 속도
drone.get_battery() - 배터리 잔량
drone.get_position_x() - X 좌표
drone.get_position_z() - Z 좌표
drone.get_distance_to(x, z) - 목표까지 거리
print(값) - 콘솔 출력
