# 초등학생 드론 코딩 교육 연간 지도계획안

## 개요

| 항목 | 내용 |
|------|------|
| **운영 목표** | 드론의 원리와 구조를 이해하고, 시뮬레이터 및 실제 드론을 통한 코딩 능력을 단계적으로 습득한다 |
| **주요 교육 내용** | 드론 기초이론, 시뮬레이터 조종, 블록 코딩, JavaScript 코딩, 실제 드론 조종, Python 코딩, 자율비행 프로그래밍 |
| **주요 강의 방법** | 이론 강의, 시뮬레이터 실습, 코딩 실습, 팀 프로젝트, 미션 수행, 발표 및 시연 |
| **운영 방법** | 주 1회 수업 / 총 48주 / 전반기(1~24주) 시뮬레이터 기반 / 후반기(25~48주) 실제 드론 기반 |
